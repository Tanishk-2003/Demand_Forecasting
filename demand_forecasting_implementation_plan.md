# Demand Forecasting Project Implementation Plan

This plan details the design and implementation of a professional, enterprise-grade demand forecasting system featuring a **React (Vite + TS + TailwindCSS)** frontend and a **FastAPI** backend, centered around the LightGBM forecasting model defined in [model.py](file:///c:/Users/tanishk.patil/Demand_Forecasting/model.py).

## User Review Required

### 2. Backend (FastAPI)

#### [NEW] [main.py](file:///c:/Users/tanishk.patil/Demand_Forecasting/backend/app/main.py)
- Setup FastAPI server with CORS middleware enabled for frontend communication.
- Define HTTP endpoints:
  - `GET /api/dimensions`: Returns unique list of Channels, Customer Segments, Categories, and Months.
  - `GET /api/forecast`: Returns aggregated actuals vs. forecast values with filters, plus confidence intervals (using historical standard deviation of errors per segment).
  - `GET /api/metrics`: Returns overall error rates (MAE, MAPE, sMAPE) and granular slices by Channel/Segment/Category.
  - `GET /api/insights`: Returns statistical and narrative explanations (e.g. top drivers, channel growth, drop anomalies).
  - `POST /api/run-forecast`: Re-runs the model pipeline and outputs new files.

#### [NEW] [forecast.py](file:///c:/Users/tanishk.patil/Demand_Forecasting/backend/app/forecast.py)
- Modularize the LightGBM multi-step backtesting and forecasting logic from `model.py` so it can be invoked programmatically.
- Save outputs into `data/outputs/` directory.

#### [NEW] [insights.py](file:///c:/Users/tanishk.patil/Demand_Forecasting/backend/app/insights.py)
- Implement a rule-based engine to scan `sales_data.csv` and `blender_forecast_next3m.csv` to find:
  - Top 3 growth channels.
  - Channels showing declining invoice/quantity counts.
  - Top segment/category combinations driving future volume.

### 3. Frontend (React + TS + TailwindCSS + Recharts)

#### [NEW] [Frontend Directory](file:///c:/Users/tanishk.patil/Demand_Forecasting/frontend/)
Create a React SPA in Vite:
- **Executive Dashboard (Page 1)**:

  - Main Trend Chart: Historical actual quantities + next 3-month forecast line with shaded confidence bands.
  - Channels overview: Horizontal bar charts of demand per channel.
- **Forecast Explorer (Page 2)**:
  - Dynamic filters sidebar: Channel, Category, Customer Segment.
  - Interactive multi-line chart comparison.
  - Download forecast data as CSV/JSON.
- **AI Insights (Page 3)**:
  - Beautiful cards summarizing "Drivers of Demand", "Key Risks", and "Anomalies".
  - Rich narrative copy explaining forecast directions (e.g., MT vs GT channel splits).
- **Model Performance / Explainability (Page 4)**:
  - Historical backtest evaluation charts showing Actual vs Backtest values across the 6 rolling origins.
  - LGBM Feature Importance chart showing weight of ASP, Customers, Invoices, Lags, etc.
